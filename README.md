# 🍔 QFood - Website Đặt Đồ Ăn Trực Tuyến

## Giới thiệu

QFood là website bán đồ ăn trực tuyến được xây dựng bằng PHP và MySQL. Hệ thống cho phép người dùng xem thực đơn, đặt món, quản lý giỏ hàng và thanh toán. Ngoài ra còn có trang quản trị dành cho Admin để quản lý sản phẩm và đơn hàng.

Đây là đồ án môn **Lập trình Web** được phát triển nhằm áp dụng các kiến thức về PHP, MySQL, HTML, CSS, Bootstrap và JavaScript.

## Cài đặt

### Bước 1

Cài đặt XAMPP.

### Bước 2

Chép project vào thư mục

```
C:\xampp\htdocs\
```

### Bước 3

Khởi động

- Apache
- MySQL

### Bước 4

Mở phpMyAdmin

```
http://localhost/phpmyadmin
```

### Bước 5

Tạo database

```
web_ban_do_an
```

Import file

```

```

### Bước 6

